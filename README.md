# Coding-practice 🧑‍💻

A collection of beginner-level coding programs written in C, C++, Java, and Python. This repository helps me practice and improve my problem-solving skills as I learn programming step by step.

## 📋 Project Overview

This is my personal coding practice repository where I work through foundational programming problems across multiple languages. Each file is a standalone program targeting a specific concept or challenge.

## 🗂️ Languages & Topics

- **Java** – OOP fundamentals, arithmetic, string handling, utility programs
- **C** – Low-level operations, control flow, pointers (coming soon)
- **C++** – Classes, loops, conditionals (coming soon)
- **Python** – Scripting, data manipulation (coming soon)

## 📁 Programs Included

- AverageMarks.java – Calculate the average of given marks
- CalculateAge.java – Determine age based on birth year
- ConvertKmToMiles.java – Unit conversion program
- More programs added regularly...

## 🧑‍💻 How to Run

For Java files:
1. Compile: javac ProgramName.java
2. Run: java ProgramName

For C/C++ files:
1. Compile: gcc filename.c -o output
2. Run: ./output
